# Social Media Skills Next Steps

Status: draft skill planning only.

Future skills may help AG OS prepare social media operating work, but skills do not grant permission and cannot authorize live action.

## Candidate Draft Skills

Potential future draft skills:

- client configuration review
- platform account readiness review
- content calendar completeness review
- post package quality review
- weekly report structure review
- blocked action safety review

## Boundaries

Skills must not:

- connect social accounts
- request or store credentials
- post content
- schedule content
- send direct messages
- comment on platforms
- pull analytics APIs
- activate n8n
- approve live actions
- promote candidate lessons into accepted lessons

## Promotion Rules

A draft skill can only become active through a separate reviewed PR. A skill remains advisory and cannot bypass the Constitution, approval locks, action matrix, cost rules, security rules, or owner authority.
