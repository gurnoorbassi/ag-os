# Credential References

This directory is reserved for credential reference records only.

Never store secret values, passwords, OAuth tokens, API keys, recovery codes, private keys, or connector credentials here.

Allowed records may only describe where a future approved credential would live, what it is allowed to do, how it is revoked, how it rotates, and what audit evidence is required.

Credential values must stay out of the repository, chat, dashboard data, logs, audit records, cost records, and client records.
