# Social Posts

This folder stores source-controlled social post records only after owner approval.

Current state: no individual live-post records are created here. The 21 AG Digitalz posts remain approved draft content through `.codex/client-management/content-sprints/content-sprint-ag-digitalz-first-content-sprint-v1.json`.

Rules:
- Draft approval is not posting approval.
- Single-post publishing requires an exact post approval lock.
- Scheduling requires a separate approval.
- Analytics requires a separate approval.
- Credentials, tokens, passwords, API keys, OAuth codes, private client data, and production customer data are forbidden.
