# Client Management

This directory is reserved for future active Client/Engagement Records v1 source-of-truth records.

No active client, engagement, deliverable, access request, or client approval records are created by this foundation. Use `.codex/templates/client-management/` until the owner approves real client data handling and active record creation.

Credentials, social access, scheduling access, analytics access, paid tools, production/customer data, and live account actions must never be stored here.
