# .codex Operating Folders

This folder stores AG Digitalz OS operating artifacts. The foundation phase tracks folder contracts, production-clean templates, empty or metadata-only registries, and policy records.

Do not store credentials, customer data, production exports, or live service payloads here.

New folders should start with README contracts before they contain records. Records that require approval, owner identity, audit history, live services, customer data, production data, paid actions, or deployments must wait for an explicitly approved scope.
