# Memory

Durable AG Digitalz OS memory records live here.

Memory must distinguish verified current facts from older notes. Do not store secrets or private customer data.

Unified Memory + Learning OS v1 uses:

- `registry.json` for the canonical memory registry.
- `accepted/` for owner-approved accepted lessons.
- `lessons/candidates/` for advisory candidate lessons.
- `rejected/` for rejected lesson records.
- `conflicts/` for promotion blockers.

Memory records never grant live-action permission.
