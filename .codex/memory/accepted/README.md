# Accepted Lessons

Accepted lessons are owner-approved, source-backed operating knowledge that AG OS workers may load at runtime.

Accepted lessons remain advisory. They do not create approval locks, authorize live actions, store credentials, override the Constitution, or bypass Security OS, Cost OS, Quality OS, approval workflow, or owner authority.

Every accepted lesson record must validate against `schemas/lesson.schema.json`, use `status: "accepted"`, include promotion evidence, and set runtime permission flags to false.
